数据类型

JavaScript 有以下几种数据类型：

  • 字符串（String）
  • 数值（Number）
  • 布尔值（Boolean）
  • 数组（Array）
  • 对象（Object）
  • 函数（Function）

字符串

字符串用单引号或双引号括起来，可以包含任意文本。

数值

数值可以是整数或浮点数。

布尔值

布尔值只有两个值：true 和 false。

数组

数组是一组按顺序排列的元素。

对象

对象是一组属性和值。

函数

函数是 JavaScript 中用于执行特定任务的块代码。

运算符

运算符是用于执行特定操作的符号。

运算符 描述 示例
+ 加法 x + y
- 减法 x - y
* 乘法 x * y
/ 除法 x / y
% 取模（余数） x % y
** 乘方 x ** y
== 等于 x == y
=== 全等 x === y
!= 不等于 x!= y
!== 不全等 x!== y
> 大于 x > y
< 小于 x < y
>= 大于等于 x >= y
<= 小于等于 x <= y
&& 逻辑与 x && y
|| 逻辑或 x || y
! 逻辑非 !x

条件语句

条件语句是用于基于特定条件执行代码的语句。

循环语句

循环语句是用于重复执行代码的语句.

事件处理程序

事件处理程序是用于响应用户事件的 JavaScript 代码。

JavaScript this 关键字

面向对象语言中 this 表示当前对象的一个引用。

但在 JavaScript 中 this 不是固定不变的，它会随着执行环境的改变而改变。

在方法中，this 表示该方法所属的对象。
如果单独使用，this 表示全局对象。
在函数中，this 表示全局对象。
在函数中，在严格模式下，this 是未定义的(undefined)。
在事件中，this 表示接收事件的元素。
类似 call() 和 apply() 方法可以将 this 引用到任何对象。
